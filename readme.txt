This plugin allows you to create domain search widgets and check domain availability from the admin dashboard.


== Description ==

Allow users to check domain name availability directly from your website.  Availability is shown without redirecting users.  With the use of widgets you can add the domain search to multiple locations on your website.  You can also quickly check domain availability yourself using the admin dashboard widget.  This plugin is multisite compatible and free.

== Installation ==

Normal plugin installation.


= Domain Search Widget =

1. Activate the widget via the "Appearance" menu in WordPress.
2. Adjust options to your liking.
3. You're Done!


== Screenshots ==

1. Search for available domains directly from your website.
2. Instantly know if a domain is available without leaving your website.
3. When a domain is available your users can choose to buy it from whichever registrar you selected in the widget options.  The registrar's website opens in a new tab.
4. WordPress Domain Search plugin creates a widget with lots of customization options.
5. Quickly check domain availability through your prefered registrar from the admin dashboard widget.
